# Architecture Decision Records

An ADR is recorded only when a decision is **hard to reverse**, **surprising
without context**, and **the result of a real trade-off**. If any of the three is
missing, it does not get an ADR.

Use [`adr-template.md`](adr-template.md) for new records.
